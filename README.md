# Computer Vision Kenya 🇰🇪

## Overview

Computer Vision Kenya is an open-source community focused on advancing Computer Vision, Deep Learning, and Visual AI technologies across Kenya and Africa.

The community brings together students, researchers, engineers, founders, and AI enthusiasts to collaborate on practical, high-impact projects that solve real-world African challenges using artificial intelligence.

This repository serves as the central hub for:

* Community projects
* Research experiments
* Open datasets
* Tutorials and educational resources
* AI tools and applications
* Weekly build challenges

---

# Mission

Our mission is to build a strong African ecosystem for Computer Vision by:

* Encouraging open-source collaboration
* Building locally relevant AI systems
* Creating educational opportunities
* Supporting innovation and research
* Developing production-ready AI applications

# Weekly Project Initiative

Computer Vision Kenya operates a weekly build system where the community collaborates to design and ship at least one Computer Vision project every week.

Examples include:

* Real-time object detection systems
* License plate recognition
* Traffic analytics
* Plant disease detection
* AI-powered attendance systems
* Smart retail monitoring
* Document intelligence systems

# Getting Started

## Prerequisites

Ensure you have the following installed:

* Python 3.10+
* Git
* pip
* Virtual environment tool

## Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/computer-vision-kenya.git
cd computer-vision-kenya
```

## Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Linux/macOS:

```bash
source venv/bin/activate
```

Windows:

```bash
venv\Scripts\activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```
